# @amlplugins/together-files

Native `together-ai` SDK re-exported under the `@amlplugins` namespace.

> Together Files — upload, list, retrieve files for Batch jobs and Fine-Tuning datasets.

## Install

```bash
npm install @amlplugins/together-files
```

## Use

```ts
import { /* named exports from together-ai */ } from "@amlplugins/together-files";
// or
import sdk from "@amlplugins/together-files";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
